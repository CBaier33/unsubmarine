# unsubmarine

## Install

```
git clone https://github.com/CBaier33/unsubmarine

cd unsubmarine
mkdir data
```

## Intialize
```
docker compose up -d --build
```

## Usage
```
http://localhost:3001/unsubscribe?email=user@example.com
```

## Customization

You can edit the `unsubmarine.html` file in anyway you'd like. In the future, a config file will be added and perhaps some other options for templates.

## About

In an effort to meet all necessary requirements to validate my email server.
